## 二八效应

80/20法则（The 80/20 Rule），又称为帕累托法则，幂律分布。

意味着每件事都有重点，做事情一定要抓重点，持续地抓住重点，就抓住了最高效率的关键。

## 头部效应

从价值而非优势出发；思考差异化优势；从最近的头部做起，从鸡头变成凤头。

通过观察和判断，抢占高价值、有优势的头部，然后从小头部走向大头部。

## TFT策略

TFT（Tit for Tat，以牙还牙）策略是解决“多重囚徒困境”最好的策略

策略就两条：第一步，合作；以后每一步，重复对手的行动——你合作我合作，你背叛我背叛。

TFT用一个最简单的方式鼓励别人和你实现共赢：善良（第一步总是表达善意）、可激怒（你背叛我背叛）、宽容（既往不咎，面向未来，你合作我合作）、简单（规则简单，让人看透）。

## 要事第一

时间管理四象限法是美国管理学家科维提出的一个时间管理的理论，也叫艾森豪威尔矩阵，把工作按照重要和紧急两个不同的维度分为四个象限。

```text
＋ 重要性
｜
｜————————————————————————————
｜　重要但不紧迫　|　紧急且重要　　｜
｜　（制定计划）　|　（立刻处理）　｜
｜———————————————————————————｜
｜　不紧急不重要　|　重要不紧急　　｜
｜　（尽量别做）　|（交他人处理）　｜
｜———————————————————————————————＋ 紧迫性
```

处理顺序：紧急又重要 -> 重要不紧急的 -> 紧急不重要 -> 不紧急也不重要

处理方式：

+ 紧急又重要：立刻去做，并尽量不要让事情拖到这个象限
+ 重要不紧急的：按计划进行，并尽量不要让事情拖到紧急
+ 紧急不重要：交给别人立刻去做，适当放权
+ 不紧急也不重要：尽量别做，劳逸结合除外

## 目标管理

目标管理由管理学大师 Peter Drucker 于 1954 年在其著作《管理实践》中提出，好的目标可以促进时间管理，SMART 原则用于衡量目标，其核心实际上就是量化。

+ S：具体（Specific），目标应该是明确的
+ M：可度量（Measurable），目标应该有指标或数据可衡量
+ A：可实现（Attainable），目标必须是在付出努力的情况下可以实现的
+ R：相关性（Relevant），目标应与其他目标具有一定的相关性，独立存在的目标无意义
+ T：时限性（Time-bound），目标必须有确定的完成时间限制

目标的分解：长期目标与短期目标；整体目标分解成小目标；

## 高效时间管理

- 重复任务自动化，工作输出标准化
- 充分利用资源：指派、授权、外包、找人跑腿，清晰告知时间要求，定期跟踪检查
- 克服拖延症，现在就做
- 文件和物品整理，减少查找时间
- 重要文件及时备份，
- 墨菲定律，事先周密计划，预留一定的时间，建立预警机制，提前想好应对策略和解决办法；
- 一次把事情做对，防止返工
- 避开完美主义，完美主义实际上是对失败的恐惧
- 使用高新技术知识和工具提高效率
- 专注，掌握时间主动权，避免任务切换带来的注意力和时间消耗；
- 学会拒绝，婉拒别人的不合理要求。
- 小事杂事集中处理，零碎时间做零碎的事
- 避免不必要的干扰：上下级、同事、减少无效信息源（各种网络应用）

## STAR 法则

STAR 法则是一种讲述自己故事的方式，通常用于面试和简历书写，对于面试官来讲，能够更精确地预测面试者未来工作表现，对于应聘者来讲，写简历和面试时能更好的体现出自己的经历。

STAR 是情境（Situation）、任务（Task）、行动（Action）、结果（Result）的缩写，具体说明如下：

+ Situation：情境，指应聘者参与过的某项事件的背景，通常问当时的情况是什么样的；
+ Task：任务，指应聘者为完成该事件所承担的工作任务，通常问当时的任务是什么，任务是谁分配的，是如何分配的；
+ Action：行动，指应聘者为完成任务所采取的行动，通常问为完成这个任务做了什么，是怎么做的；
+ Result：结果，指应聘者完成任务后得到的结果，通常问这项任务是否完成目标，得到了什么。

STAR 法则实际上是一种结果导向思维。

## MECE 原则

MECE (发音：me see) 原则是巴巴拉·明托 (Barbara Minto) 在金字塔原理 (The Minto Pyramid Principle) 中提出的一条结构化思维原则。

MECE 全称是 Mutually Exclusive，Collectively Exhaustive，通常翻译成“相互独立，完全穷尽”，简而言之，不重不漏。

MECE 的本质是选择一种合适的分类方法，当我们要处理复杂问题时，如果分类没有覆盖问题的所有方面，那我们的论证就可能是不全面的，最终结论也可能是不可靠的。MECE 的难点在于，在确定问题的范围内，如何寻找符合 MECE 的切入点。

## 5W2H分析法

5W2H分析法又叫七问分析法，由二战中美国陆军兵器修理部首创。

+ What：目标，是什么？目的是什么？做什么工作？

+ Why：原因，为什么要做？可不可以不做？有没有替代方案？

+ Who：参与/执行者，谁？由谁来做？

+ When：时间，何时？什么时间做？什么时机最适宜？

+ Where：地点，何处？在哪里做？

+ How：方式，怎么做？如何提高效率？如何实施？方法是什么？

+ How Much：指标，要多少产出？做到什么程度？质量水平如何？费用如何？

## 思维导图

思维导图是英国人东尼·博赞 (Tony Buzan) 创建的一个思维梳理和发散工具。

1. 绘制原则

    - 中心扩散
    - 粗线条描绘分支
    - 只用关键字
    - 多用画图方式表达
    - 善用色彩

2. 误区

    思维导图本身是一个梳理思维和辅助记忆的工具，重点是思维，核心在于关键的词语和准确的结构，如果不能抓住这个核心，就容易陷入误区。

    - 过度绘制：要避免花费大量时间在色彩填充和分支布局上，这样不只起不到梳理思维的作用，反而浪费不少时间。
    - 没有逻辑：不少人喜欢用思维导图记笔记，结果绘制的思维导图要么是一系列知识点的罗列，要么和图书目录差不太多，这些都是只有图没有思维的体现。

## 头脑风暴

头脑风暴 (Brain Storming) 是指一群人围绕一个特定的兴趣领域，在融洽和不受限制的氛围中进行讨论并产生新观点。

头脑风暴主要基于，在一个不受限制的讨论环境下，人会因竞争意识与表达欲望而积极思考与发言，一方面，每提出一个新的观念，都能引发他人的联想，相继产生一连串的新观念，另一方面，相互感染会形成热潮，能进一步促进思维。

1. 要点

    - 自由畅谈
    - 延迟评判
    - 禁止批评
    - 以量求质

2. 纸上头脑风暴

    也叫书面头脑风暴法，所有人都先在纸上写好自己的观点，然后相互交互，了解别人的观点后，写下更多的观点。

    书面头脑风暴避免了参与者因地位和性格等方面带来的影响，但由于参与者都是自己思考，激励不充分。


## 假设逆转

是一种逆向思维，其步骤为

1. 考虑与问题相关的产品、概念、特征特点等相关要素；
2. 将要素逆转
3. 使逆转后的要素可实现

假设逆转法的核心在于想办法使逆转后的要素可实现，举个例子：如何使菜馆盈利？

| 要素       | 逆转       | 新观点     |
| ---------- | ---------- | ---------- |
| 有客人就餐 | 没客人就餐 | 只做外卖   |
| 有好厨师   | 没好厨师   | 食材标准化 |

## 图片联想

是一种通过图片、借用联想和类比，启发思维解决问题的方法。其步骤为：

1. 选择一组随机图片
2. 抛开要解决的问题，观察图片、产生联想，列出要点
3. 回到问题，根据上述线索，强制联想，产生解决问题的方法。

举个例子：如何让菜馆盈利？随机选择一张图片，比如牛，那么产生的线索和解决问题的方法类似如下：

| 线索     | 解决方案：如何让菜馆盈利？ |
| -------- | -------------------------- |
| 吃苦耐劳 | 不怕辛苦                   |
| 庖丁解牛 | 对客户和系统更了解         |
| 股市上涨 | 做好客户，股票会上涨       |

## 最不可能

通过如何做到最不可能来激发创意，其步骤为：

1. 考虑与问题相关的产品、概念、特征特点等相关要素；
2. 根据具体情况进行极限假设；
3. 根据极限假设进行修改，使其具有可实现性。

举个例子：如何使餐馆盈利？

| 极限假设         | 解决方案 |
| ---------------- | -------- |
| 一点完菜就做好   | 减少选择，事先准备 |

## 反脆弱

反脆弱的能力，就是在经历小的刺激和压力并恢复后变得更加强大，比如人的免疫系统。创新往往是一群人为解决某个具体的问题产生的，缺乏创新可能是面对的麻烦不够多。适量的刺激和麻烦，可以让人和组织更强悍，更有创造力。

## 多次收益与长尾收益

给别人打工，提供的是一个交付件，只有一次受益，事实上这个交付件可能会使多个人受益，即多次收益，这个交付件会持续带来收益，即长线收益的：一个人要从消费者向生产者转变，经营点自己的东西，自己交付，自己拿多次收益和长线收益。

## 不给自己设限

资源边界越小，越容易跟其他人重合，越容易引发竞争；资源边界越大，竞争越小，想的是进一步扩大资源边界，越容易促进合作；这个世界的资源是无穷的，太执着于眼前的好处，可能会忽略世界很大，资源很多。不要给自己设限，学习新东西，拓展新边界。

## 内卷化需要外部资源

内卷化指一种社会（文化模式）某一发展阶段达到某种确定的形式后便停滞不前，难以转化为高级别的模式，从而把自己锁死在低水平的状态，周而复始地循环。解决内卷化一是要注入外界资源，二是要增加不确定性。对个人而言，教育是一个人的贵人，是解决一个人内卷化的最好外界资源。

## 增加不确定性

增加不确定性就要多试错，在能承受的范围内多尝试，多去面对不确定性，运气才能开始出现。

## 负利率意味着机会耗尽

利率和经济增长是正相关的，经济不好的时候，没什么可以投资的项目，利率就非常低。负利率的出现意味着市场盈利机会的耗尽，颇破局的关键在发现额外的蛋糕。
